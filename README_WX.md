===============

## 1.微信公众账号介绍和申请流程
* 微信应用——微信App
* 微信用户
* 微信公众号——自媒体
    * 服务号——企业和组织，不适用于个人
    * 订阅号——个人和组织
    * 企业号——针对企业

## 2.微信公众号工作原理和接口介绍
* 工作原理：用户->微信服务器->三方服务器
* 接口介绍：
    * 基础接口
        * 事件推送——订阅公众账号
        * 消息响应——发送普通消息，接受普通消息
        * 基础接口——获取access_token，获取微信服务器地址
## 3.在应用中接入微信公众账号API
* 接入微信api验证
    *  1 token，timestamp，nonce按字典序排列
    * 2 将排序后的三个参数拼接之后用sha1加密
    * 3 将加密后的字符串和将signature进行对比，判断该请求是否来自微信
